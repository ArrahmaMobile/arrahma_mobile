   - Lesson 1: Surah Al-Fatiha Ayah 1 (11/26/25) - Audio
   - Lesson 2: Surah Al-Fatiha Ayah 2-5 (12/5/25) - Root words PDF + Multiple audio files
   - Lesson 3: Surah Al-Fatiha Ayah 5-6 (12/13/25) - Root words PDF + Multiple audio files

   **Surah Al-Baqarah - البقرۃ**
   - Lesson 4: Surah Al-Baqarah Ayah 1-5 (12/19/25) - Root words PDF + Multiple audio files
   - Lesson 5: Surah Al-Baqarah Ayah 6-13 (1/2/26) - Root words PDF + Audio files
   - Lesson 6: Surah Al-Baqarah Ayah 14-20 (1/9/26) - Root words PDF + Audio files
   - Lesson 7: Surah Al-Baqarah Ayah 21-25 (1/16/26) - Root words PDF + Audio files

**Audio File Naming Patterns:**
- Main lecture: `https://arrahmah-media.org/taf2025/juz1/[date]-[surah][ayah].mp3`
- Women's class: Files marked with `(wm)` suffix
- Parts for longer lectures: `part1(t)`, `part2(t)` suffixes
- Root words PDFs: `../root_words/[year]/r[number].pdf`

## 11. RECOMMENDATIONS FOR SCRAPER

1. **Handle Multi-Level Menus:** Tafseer menu has 2 levels of nesting
2. **Track Course Status:** Active vs. Closed registration
3. **Multiple Audio Files:** Some lessons have 2-3 audio files
4. **Language Variants:** Same course in different languages
5. **External Resources:** Google Forms, YouTube, external domains
6. **Date Formats:** Various date formats used (MM/DD/YY, Month DD, YYYY)
7. **Arabic Text:** Preserve Arabic lesson titles and Quranic text
8. **PDF Types:** Distinguish between course details, practice words, root words, reference materials
9. **Dynamic Content:** Carousel and ticker content changes
10. **Resource Availability:** Some lessons marked as "coming soon"
